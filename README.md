# Overview

This website serves as a central portfolio hub for projects involving:

- Scientific computing
- Numerical methods
- Computational physics
- Data visualization

# Current Projects Featured

## PyVelora
A scientific computing library focused on linear algebra, differential equations, and computational physics workflows.

## Algorithm Visualizer
A Python GUI project for visualizing sorting and searching algorithms using generator-based state systems.

## Scientific Data Processor
A FastAPI + JavaScript application for uploading and processing scientific datasets.

## ODE Solver Experiments
Experimental numerical and matrix-based approaches for solving ordinary differential equations.

# Running Locally

Clone the repository:

```bash
git clone https://github.com/njryan-boou/portfolio-website.git
```

Open the project folder:

```bash
cd portfolio-website
```

Then open:

```bash
index.html
```

You can also use the VS Code Live Server extension for local development.
